Часть 1.
    1. Написать функцию преобразования изображения из цветового пространства RGB в цветовое пространство YCbCr.
    Написать функцию обратного преобразования.
    Входные и выходные данные - массив размера (N,M,3) (или любые структуры, поддерживающие аналогичное разбиение) типа int8.
    2.
    Написать функцию обхода прямоугольной матрицы зигзагом.
    Входные данные - двумерный массив размера (N,M), выходные данные - одномерный массив длины N*М.
    3. Написать функцию даунсэмплинга изображения (отдельного канала изображения) тремя способами:
    - удалением строк и столбцов;
    - заменой блока пикселем с средним значением цвета блока;
    - заменой блока на пиксель, ближайший по значению к среднему.
    Входные данные - двумерный массив размера (N,M), коэффициенты сжатия по каждой оси Сх и Су, способ Даунсэмплинга.
    Выходные данные - двумерный массив размера (N/Cx, М/Су).
    4. Написать функцию амсэмплинга изображения с заменой одного пикселя на блок пикселей с тем же значением цвета.
    Входные данные - двумерный массив размера (N,M), коэффициенты расширения по каждой оси Сх и Су.
    Выходные данные - двумерный массив размера (N*Cx, М*Су).

Часть 2.
    1. Написать функцию двумерного дискретного косинусного преобразования.
    Входные данные: Массив компоненты цвета пикселей размера (8,8) типа int8.
    Выходные данные: Массив коэффициентов ДКП размера (8,8) типа int8.
    Написать функцию обратного двумерного дискретного косинусного преобразования.
    2. Написать функцию, возвращающую матрицу квантования для заданного уровня качества.
    Входные данные: уровень качества Q от 1 до 100.
    Выходные данные: Массив значений матрицы квантования размера (8,8).
    3. Написать функцию квантования матрицы коэффициентов ДКП.
    Входные данные: матрица коэффициентов ДКП, матрица квантования.
    Написать функцию деквантования матрицы ДКП
    4. Написать вариант RLE для последовательности квантованных коэффициентов матрицы ДКП, полученных с помощью обхода зигзагом.
    Подумать об эффективном варианте записи коэффициентов, предполагая наличие в последовательности множества нулей.
    Исследовать влияние уровеня качества на качество изображения используя последовательность: ДКП->квантование->деквантование->обратное ДКП

Часть 3.
    Реализовать алгоритм сжатия, инспирированный стандартом сжатия JPEG с следующими основными этапами:
    1. Преобразование в цветовое пространство УСЬСГ
    2. Сабсэмплинг
    3. Дискретное косинусное преобразование
    4. Квантование
    5. Энтропийное кодирование

    Удостовериться в корректной декомпрессии изображения.

    1. Исследовать эффективность сжатия на следующих тестовых данных:
    Цветное изображение из первой лабораторной работы;
    Изображение, в котором все пиксели имеют один цвет;
    Изображение, в котором цвет каждого канала RGB является случайной дискретной величиной, имеющей равномерное распределение на отрезке [0,255].
    1.1. Оценить вклад каждого этапа в коэффициент сжатия при параметре качества 50 и свести их в таблицу.
    1.2. Исследовать зависимость размера сжатого файла от параметра качества, свести результаты в виде коэффициента сжатия для различных тестовых изображений и различных коэффициентов качества.
    
    2. Получить оценку функции сложности алгоритма.
    
    Отчет по лабораторной работе должен содержать:
    Экспериментальную часть с результатами исследования эффективности сжатия, как всего алгоритма в целом, так и отдельных частей;
    Примеры исходных и сжатых изображений;
    Выводы по полученным результатам;
    Ссылку на репозиторий Github с исходным кодом;
    Раздел со всем написанным кодом. Каждая функция должна иметь заголовок.
    Отчет может не содержать теоретическую часть.
